Web 1.0, Web 2.0 and Web 3.0

Web 1.0 : The first version of web also referred as Syntactic web or read only web is the era(1990 – 2000) where the role of a user is limited to reading information provided by the content producers. There is no option given for user or consumer to communicate back the information to the content producers.

Web 2.0: also referred as Social Web or read-write web is the era(2000 – 2010 and continues even now) which facilitates interaction between web users and sites which intern allows users to communicate with other users. In this era every user can be a content producers and content is distributed and shared between sites.

Web 3.0: also referred as Semantic Web or read-write-execute is the era(2010 and above) which refers to the future of web. In this era computers can interpret information like humans via Artificial Intelligence, Image recognition and Machine Learning. Which help to intelligently generate and distribute useful content tailored to a particular need of a user.

Three main differences:

Web 1.0 :
1) Mostly Read-Only
2) Company Focus
3) Owning Content

Web 2.0 :
1) Wildly Read-Write
2) Community Focus
3) Sharing Content

Web 3.0 :
1) Portable and Personal
2) Individual Focus
3) Consolidating Content

Differences in Web 1.0 2.0 and 3.0 with with the example of searching for Medication Side Effects:

Web 1.0:
We would be directed to a page where the answer will be plain text or maybe a link, no high-level of interaction.


Web 2.0:
We would get number of results from a known search engine, maybe answers from YouTube video explanations or forum threads such as Reddit. We could also ask in social communities such as Facebook, special medical forums etc.


Web 3.0:
We would get results based on out preferences, preferred sites and type of content, Specially tailored information that the web has selected for us.
